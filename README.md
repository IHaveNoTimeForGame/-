# shadiao_tool
文字自动配图工具
在这个中文互联网内容氛围都被带向了奇怪的方向的时代。
博客文案如果没有沙雕表情包，都没办法看下去。
每次写完博客和文案都需要配图加表情包，好麻烦。
那如果有工具可以自动添加那该多好!
于是乎，本着外事问谷歌，内事问百度，某事问天涯的原则，上网搜索了一通，没有发现有此类的工具，可以直接使用。
无奈只好自己动手丰衣足食，自己撸一个。
大概思考了片刻，确定了思路
无外乎就是对一篇文章分段，确定多少文字配一张表情包。
提取每一个分段的关键字，确定放什么表情包。
然后把文字和图片混合就行。

关键词提取那就用Jieba。老厉害的分词，提取关键词工具。
获取表情包图片就用万能的爬虫。
图文混合，想到的办法就是造word文档。估计大家经常会看到一些视频讲程序操作excel的。但是其实office全家桶都可以用程序控制。我们就是用py-docx来生成编辑word文档。
为了使用方便，把程序进行个打包。打包那当然用pyinstaller啦。
